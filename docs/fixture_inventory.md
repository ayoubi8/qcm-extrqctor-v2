# Baseline Fixture Inventory

Formal golden datasets are deferred, but Plan 01 reserves the structure for fixtures and regression evidence.

## Initial Fixture Buckets

- `fixtures/source_pdfs/`: source PDFs under the 50 MB application cap.
- `fixtures/legacy_outputs/`: read-only legacy step output samples.
- `fixtures/artifacts/`: canonical artifact payload examples by schema version.
- `fixtures/security/`: two-user denial and ownership scenarios.

## Current Status

No production fixtures have been copied into the new implementation workspace yet. Later plans should add small, non-sensitive examples first, then larger manual validation datasets when approved.
